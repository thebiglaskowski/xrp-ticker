# /debt - Technical Debt Tracker

Load and execute: `C:\scripts\prompts\operations\TECH_DEBT_TRACKER.md`

## Quick Start

Identify and catalog technical debt.

**Input needed:**
- Codebase or specific area
- Known pain points (optional)

**Output:**
- Debt inventory
- Impact assessment
- Prioritization matrix
- Paydown recommendations

**Use when:**
- Sprint planning
- After shipping features
- When velocity drops
