# /adr - Architecture Decision Record

Load and execute: `C:\scripts\prompts\documentation\ADR_WRITER.md`

## Quick Start

Document architectural decisions.

**Input needed:**
- Decision to document
- Context and constraints
- Options considered

**Output:**
- Formatted ADR document
- Decision rationale
- Consequences documented

**Use when:**
- Making technology choices
- Changing architecture
- Recording why something was built a certain way
