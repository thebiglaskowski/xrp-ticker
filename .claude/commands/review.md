# /review - Code Review

Load and execute: `C:\scripts\prompts\quality\CODE_REVIEW.md`

## Quick Start

Structured code review before merging.

**Input needed:**
- Code to review (files, PR, or diff)
- Context (what it's supposed to do)

**Output:**
- Review findings by category
- Severity-rated issues
- Actionable feedback
- Approval/changes-requested verdict

**Use when:**
- Before merging PRs
- Self-review before commit
- Reviewing inherited code
