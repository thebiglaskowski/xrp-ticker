# /audit-blueprint - Blueprint Validation

Load and execute: `C:\scripts\prompts\planning\BLUEPRINT_AUDITOR.md`

## Quick Start

Validate a project blueprint before execution begins.

**Input needed:**
- Blueprint document(s) to audit
- CLAUDE.md (if exists)

**Output:**
- Structural coverage mapping
- Red-team findings with severity ratings
- Required planning artifacts
- Execution-readiness score (0-100)

**Use when:**
- Before starting implementation
- After major spec changes
- When inheriting an existing plan
