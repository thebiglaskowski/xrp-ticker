# /postmortem - Incident Analysis

Load and execute: `C:\scripts\prompts\operations\INCIDENT_POSTMORTEM.md`

## Quick Start

Learn from production incidents.

**Input needed:**
- Incident timeline
- Impact details
- Resolution steps taken

**Output:**
- Blameless analysis
- Root cause identification
- Action items
- Prevention measures

**Use when:**
- After production incidents
- After near-misses
- Quarterly incident reviews
