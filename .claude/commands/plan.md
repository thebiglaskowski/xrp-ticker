# /plan - Feature Specification

Load and execute: `C:\scripts\prompts\planning\FEATURE_SPEC_WRITER.md`

## Quick Start

Turn a feature request into a detailed, executable specification.

**Input needed:**
- Feature name or description
- User stories or requirements (if available)
- Constraints or dependencies

**Output:**
- Complete feature specification with acceptance criteria
- Technical approach
- Risk assessment
- Dependency map
