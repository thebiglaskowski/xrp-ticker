# /daily - Daily Build Session

Load and execute: `C:\scripts\prompts\execution\DAILY_BUILD.md`

## Quick Start

Continue development work in a focused session.

**Input needed:**
- STATUS.md (current state)
- Blueprint (what we're building)

**Output:**
- Today's focus identified
- Work completed
- Updated STATUS.md
- Updated CHANGELOG.md (if applicable)

**Use at:**
- Start of each coding session
- After context resets
- When resuming work
