# /migrate - Migration Planning

Load and execute: `C:\scripts\prompts\operations\MIGRATION_PLANNER.md`

## Quick Start

Plan safe data or schema migrations.

**Input needed:**
- What's being migrated
- Source and target states
- Constraints (downtime, data loss, etc.)

**Output:**
- Migration strategy
- Rollback plan
- Validation steps
- Risk mitigation

**Use when:**
- Database schema changes
- Data format changes
- System migrations
